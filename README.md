Hotel Revenue & Booking Analysis Dashboard | Power BI

📌 Project Summary

Developed an interactive Business Intelligence dashboard using Power BI to analyze hotel booking data, revenue trends, and customer behavior.
The project transforms raw operational data into actionable insights to support strategic decision-making in the hospitality domain.

🎯 Business Problem

Hotels often struggle to track booking performance, revenue fluctuations, and cancellation impact effectively.
This dashboard provides a centralized analytical solution to monitor KPIs and improve revenue optimization strategies.

📊 Key Business Metrics Analyzed

Total Revenue

Total Bookings

Cancellation Rate

Average Daily Rate (ADR)

Occupancy Trends

Customer Type Contribution

🔍 Key Insights Generated

Identified peak and off-season booking patterns

Measured revenue loss due to cancellations

Analyzed customer segments contributing most to revenue

Compared performance across different booking channels

Business Insight Interpretation

📈 Impact

This dashboard enables stakeholders to:
✔ Track performance in real-time
✔ Reduce revenue leakage from cancellations
✔ Make data-driven pricing and occupancy decisions
✔ Improve strategic planning using trend analysis
